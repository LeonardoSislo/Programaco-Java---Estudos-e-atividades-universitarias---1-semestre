# 🚀 Estudos de Programação Java - 1º Semestre

Este repositório foi criado para armazenar e organizar todos os exercícios e atividades desenvolvidas durante o meu primeiro semestre na faculdade. O foco principal é o aprendizado da linguagem **Java** e fundamentos de **Lógica de Programação**.

## 📚 Conteúdos Praticados

Até o momento, o repositório contém exercícios sobre:

* **Estruturas Sequenciais:** Cálculos básicos (Ex: Cálculo Salarial).
* **Estruturas Condicionais:** Tomada de decisão com `if/else` e `switch`.
* **Vetores e Matrizes:** Manipulação de coleções de dados e tabelas.
* **Algoritmos Aplicados:** Criação de calculadoras e sistemas de lista de produtos.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Java (JDK 17+)
* **Ambiente de Desenvolvimento:** (Apache NetBeans IDE 25)
* **Controle de Versão:** Git & GitHub

## 📂 Como visualizar os códigos
As pastas estão organizadas por tema. Dentro de cada uma, você encontrará o arquivo `.java` com a resolução do problema proposto em aula.
